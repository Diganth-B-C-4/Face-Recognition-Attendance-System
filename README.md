# Face Recognition Attendance System

## 📌 Overview
A Face Recognition Attendance System built using Python, OpenCV, FaceNet, MTCNN, DeepSORT, SQLAlchemy, and SQLite. The system detects and recognizes faces in real time and automatically marks attendance.

## 🚀 Features
- Face Registration
- Real-Time Face Recognition
- Automatic Attendance Marking
- SQLite Database Storage
- Face Tracking using DeepSORT

## 🛠️ Technologies Used
- Python
- OpenCV
- PyTorch
- FaceNet (InceptionResnetV1)
- MTCNN
- DeepSORT
- SQLAlchemy
- SQLite

## 📂 Project Structure

```
Face-Recognition-Attendance-System/
│── main.py
│── register.py
│── attendance.db
```

## ▶️ How to Run

### Install dependencies

```bash
pip install -r requirements.txt
```

### Register a Face

```bash
python register.py
```

### Start Attendance

```bash
python main.py
```

## 📷 Output

- Detects faces in real time.
- Recognizes registered users.
- Marks attendance automatically.
